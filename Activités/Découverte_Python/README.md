## Activités pédagogiques
  
Les activités proposées ici ont pour objectif d'aider des élèves à s'approprier le langage Python via des activités pratiques.  

Les algorithmes sont donnés sout la forme de commentaires qu'il faut "traduire" en langage Python. 
  
Ce dossier contient une fiche PDF sur des exemples d'utilisation de fonctionnalités de la carte qui pourront apporter une aide précieuse dans la résolution des exercices.  

Enjoy yourself ;-)
